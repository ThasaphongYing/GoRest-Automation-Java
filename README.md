# GoRest API Automation Testing

This project is an automated testing suite for the GoRest API using **Java**

## Setup Instructions
1. **Prerequisites**: Ensure you have JDK 11+ 
2. **Clone the repository**
3. **Get API Token**: To get your Bearer Token.
4. **Configuration**: Open `UserTests.java` and replace `YOUR_ACTUAL_TOKEN_HERE` with your real token.

## How to Run
Run the following command in the terminal:
```bash
mvn test
