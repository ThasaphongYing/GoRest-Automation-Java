# GoRest API Automation Test Suite

This project automates the testing of User Management endpoints at **GoRest API** using Java and RestAssured.

## 📌 Requirements
- Java JDK 11 or higher
- Apache Maven
- A valid Bearer Token from [GoRest.co.in](https://gorest.co.in/)

## 🚀 Getting Started
1. Clone this repository to your local machine.
2. Open the project in your favorite IDE (e.g., IntelliJ IDEA, Eclipse).
3. Open `src/test/java/com/gorest/tests/GoRestApiTest.java`.
4. Replace `YOUR_TOKEN_HERE` with your actual Bearer Token.
5. Run the tests via terminal using Maven:
   ```bash
   mvn test
